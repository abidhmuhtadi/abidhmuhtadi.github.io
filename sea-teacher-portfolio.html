Program SEA Teacher

Nama Anda Di Sini

Program Studi Pendidikan — Universitas Anda

Saya adalah mahasiswa program pendidikan yang berkesempatan mengikuti program SEA Teacher (Student Exchange and Teaching Experience in Southeast Asia) yang diselenggarakan oleh SEAMEO.

Selama program ini, saya menjalani praktik mengajar di sekolah mitra di Thailand, mengalami langsung dinamika kelas multikultural, serta mengembangkan perangkat pembelajaran inovatif yang menjadi bahan penelitian skripsi R&D saya.

Panduan Upload

🚀 Deploy ke Vercel

Ikuti 4 langkah berikut agar website ini bisa diakses secara online dan bisa Anda sertakan sebagai lampiran skripsi atau portofolio.

Panduan Deployment — vercel.com
1
Buat akun gratis di Vercel

Buka vercel.com → klik Sign Up → pilih Continue with GitHub (atau daftar dengan email). Akun gratis sudah cukup untuk portofolio ini.

2
Siapkan folder proyek Anda

Pastikan file HTML ini bernama index.html dan semua foto/aset lain ada dalam satu folder. Nama folder bisa bebas, misalnya sea-teacher-portfolio/

3
Upload ke Vercel

Di dashboard Vercel → klik Add New → Project → pilih tab "Browse" atau "Import" → drag & drop folder Anda ke area upload → tunggu hingga proses selesai (±30 detik).

4
Website Anda sudah online! 🎉

Vercel akan memberi URL otomatis seperti: nama-proyek.vercel.app Bagikan URL ini ke dosen, teman, atau sertakan di lampiran skripsi. Untuk update: cukup upload folder lagi, Vercel akan refresh otomatis.

💡 Tips untuk Mahasiswa: Jika ingin domain kustom (misal namamu.com), Anda bisa beli domain seharga Rp 50–100 ribu/tahun di Niagahoster atau IDCloudHost, lalu hubungkan ke Vercel lewat menu Domains. Namun, domain gratis dari Vercel sudah sangat layak untuk keperluan skripsi.
